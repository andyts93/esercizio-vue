# Esercizio Vue

## Setup progetto
```
yarn install
```

### Compilazione e hot-reload
```
yarn serve
```

### Compilazione e minimizzazione per release
```
yarn build
```

### Lint e sistemazione file
```
yarn lint
```

Vedi [documentazione](https://cli.vuejs.org/config/).
